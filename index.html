Connecte ta base de données

Le carnet de tournée stocke maintenant tes contacts sur Firebase (Google), une base gratuite qui t'appartient. Tes données restent accessibles même si tu recrées cette app plus tard. Ça prend 5 minutes, une seule fois.
  1. Va sur console.firebase.google.com, connecte-toi avec un compte Google et crée un projet (gratuit, sans carte bancaire).
  2. Dans le menu de gauche, ouvre Firestore Database → Créer une base de données → choisis un mode (peu importe, on changera les règles) → une région proche de toi.
  3. Toujours dans Firestore, onglet Règles, remplace le contenu par celui-ci puis Publier :
    rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /contacts/{contactId} { allow read, write: if request.auth != null; } } }
  4. Va dans Authentication → Sign-in method → active le fournisseur Anonyme.
  5. Retourne dans Paramètres du projet (roue crantée) → onglet Général → section "Vos applications" → clique l'icône </> (Web), donne un nom, ne coche pas Hosting, puis Enregistrer l'application.
  6. Copie l'objet firebaseConfig affiché et colle-le ci-dessous tel quel.
